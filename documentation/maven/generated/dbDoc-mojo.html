liquibase:dbDoc

Full name:

org.liquibase:liquibase-maven-plugin:3.8.8:dbDoc

Description:

Generates dbDocs against the database.

Attributes:

  • Requires a Maven 2.0 project to be executed.
  • Requires dependency resolution of artifacts in scope: test.
